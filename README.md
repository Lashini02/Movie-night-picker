# 🎬 Movie Night Picker – Web Application 

Movie Night Picker is a dynamic web application developed using **PHP, HTML, CSS, and JavaScript** to help users quickly and easily decide what movie to watch.  
The system allows users to add movies, manage a movie list, and randomly select a movie for a stress-free movie night experience.

This project demonstrates both **front-end and back-end web development concepts**.

## 📌 Introduction

Choosing a movie for a group often results in delays and disagreements.  
Movie Night Picker provides a simple solution by using a **server-side PHP logic** combined with a clean user interface to randomly select movies from a user-defined list.

The application runs locally using a PHP server and does not require advanced configuration.

---

## ❗ Problem Statement

- Difficulty in choosing a movie for group movie nights  
- Time wasted discussing movie options  
- Lack of simple tools without login requirements  
- No lightweight movie picker with backend logic  

---

## ✅ Solution Overview

The Movie Night Picker application:
- Uses **PHP** to handle backend logic
- Stores movie data (array or database)
- Randomly selects a movie server-side
- Displays results dynamically to the user
- Allows re-selection without page reload (optional JS support)

## 🧩 System Architecture
-User Interface (HTML/CSS/JS)
    ↓
-PHP Backend
    ↓
-Movie Logic / Database
    ↓
-Result Display

🔄 Application Workflow

-User opens the Movie Night Picker website
-User adds movies via the form
-Movies are processed using PHP
-User clicks Pick Movie
-PHP randomly selects a movie
-Selected movie is displayed
-User can re-pick if needed
